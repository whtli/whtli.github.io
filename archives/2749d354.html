LeetCode精选TOP面试题160.相交链表

2020-03-27 LeetCode 822 字
Java LeetCode 面试 数据结构 链表

题目描述

编写一个程序，找到两个单链表相交的起始节点。返回值是ListNode类型。
示例 1：

输入：intersectVal = 8, listA = [4,1,8,4,5], listB = [5,0,1,8,4,5], 
	 skipA = 2, skipB = 3
输出：Reference of the node with value = 8
输入解释：相交节点的值为 8 （注意，如果两个列表相交则不能为 0）。
		从各自的表头开始算起，链表 A 为 [4,1,8,4,5]，链表 B 为 [5,0,1,8,4,5]。
		在 A 中，相交节点前有 2 个节点；在 B 中，相交节点前有 3 个节点。

示例 2：

输入：intersectVal = 2, listA = [0,9,1,2,4], listB = [3,2,4], 
	 skipA = 3, skipB = 1
输出：Reference of the node with value = 2
输入解释：相交节点的值为 2 （注意，如果两个列表相交则不能为 0）。
		从各自的表头开始算起，链表 A 为 [0,9,1,2,4]，链表 B 为 [3,2,4]。
		在 A 中，相交节点前有 3 个节点；在 B 中，相交节点前有 1 个节点。

示例 3：

输入：intersectVal = 0, listA = [2,6,4], listB = [1,5], skipA = 3, skipB = 2
输出：null
输入解释：从各自的表头开始算起，链表 A 为 [2,6,4]，链表 B 为 [1,5]。
		由于这两个链表不相交，所以 intersectVal 必须为 0，
		而 skipA 和 skipB 可以是任意值。
解释：这两个链表不相交，因此返回 null。

要求：

  • 如果两个链表没有交点，返回 null.
  • 在返回结果后，两个链表仍须保持原有的结构。
  • 可假定整个链表结构中没有循环。
  • 程序尽量满足 O(n) 时间复杂度，且仅用 O(1) 内存。

解题思路

  • 分别遍历两个链表，统计两个链表各自的长度lenA和lenB；
  • 先对较长的链表进行遍历，向前遍历 | lenA - lenB | 次；
  • 然后利用while循环同时遍历两个链表：
      + 如果遍历过程中找到两个链表的结点完全相等，那么返回这个结点；
  + 如果遍历结束依然没有返回某个结点，那么就说明这两个链表不相交，返回null。

时间复杂度 O(m + n)
空间复杂度 O(1)

代码实现（Java）

package it160.getIntersectionNode;

import sun.nio.cs.ext.MacArabic;

/**
 * @author : flower48237
 * @2020/3/27 9:52
 * @title : LeetCode精选TOP面试题160.相交链表
 */
/**
 * 输入示例
 * 8
 *   [4,1,8,4,5]
 * [5,0,1,8,4,5]
 * 2
 * 3
 */
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        int lenA = 0, lenB = 0; // 用来记录两个链表的长度
        ListNode p = headA;		
        while (p != null){
            lenA ++;			// 得到链表A的长度
            p = p.next;
        }
        p = headB;				
        while (p != null){
            lenB++;				// 得到链表B的长度
            p = p.next;
        }
        if (lenA < lenB){		// B 更长
            for (int i = 0; i < lenB - lenA; i ++){
            	// 先对长的链表B遍历 lenB - lenA 次
                headB = headB.next;
            }
            while (headA != null && headB != null){
                if (headA == headB){
                	// 找到两个链表中完全相同的结点并返回
                    return headA;
                }
                headA = headA.next;
                headB = headB.next;
            }
        }else {					// A 更长 或者 A 与 B 等长
            for (int i = 0; i < lenA - lenB; i ++){
            	// 先对长的链表B遍历 lenA - lenB 次
                headA = headA.next;
            }
            while (headA != null && headB != null){
                if (headA == headB){
                	// 找到两个链表中完全相同的结点并返回
                    return headA;
                }
                headA = headA.next;
                headB = headB.next;
            }
        }
        return null; // 不想交
    }
}
