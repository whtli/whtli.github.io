安装Anaconda+常用的conda命令+Anaconda换源+安装Jupyter

2021-12-08 D2L 299 字
深度学习 Jupyter Anaconda conda

安装Anaconda

  • 下载并安装Anaconda

常用的conda命令

# 创建虚拟环境
conda create -n pytorch python=3.6
# 删除虚拟环境
conda env remove -n pytorch
# 激活指定虚拟环境
conda activate pytorch
# 退出当前虚拟环境
conda deactivate
# 安装包，如numpy（提前进入目标环境）
conda install numpy
或
pip install numpy
# 安装指定版本的包，如numpy
pip install numpy==1.6.0
# 升级包
pip install --upgrade numpy
# 卸载包
pip uninstall numpy
# 查看conda中所有环境
conda info -e
# 查看某个环境中安装的所有包（提前进入目标环境）
conda list

Anaconda换源

  • 修改镜像为国内源，如清华源

安装Jupyter

  • 安装Anaconda软件之后，在首页找到Jupyter点击Install。

  • 不同虚拟环境下各自的Jupyter，只需提前在Anaconda软件界面切换虚拟环境。

  • 为不同的jupyter指定不同的工作路径，首先为这个jupyter新增一个本地工作路径path，找到“开始”处新增的快捷方式：Jupyter Notebook (pytorch)，右键打开文件路径，再右键属性，修改目标(T)，把"%USERPROFILE"修改为指定路径path，如图。

  • 修改后单击该快捷方式即可进入网页使用JupyterNoteBook
